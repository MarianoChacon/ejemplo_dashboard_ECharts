Lógica de funcionamiento de un proyecto con django y bootstrap en back end y css, javascript y ECharts en el front.

En las vistas (views.py) se crea la función que devuelve un objeto Json (JsonResponse) con el gráfico en sí. Entonces los valores de ese gráfico
se los paso a esa función directamente. Luego, en el front con Javascript se toma ese objeto Json y se lo muestra en el html.
